# Syed Abubakar Siddique - Professional Portfolio

Welcome to the source code for my professional portfolio website! This project is a modern, responsive, and highly interactive web application designed to showcase my software engineering projects, technical skills, and academic achievements.

## 🌟 Key Features

- **Premium Glassmorphism UI**: A sleek, modern design aesthetic featuring frosted glass panels, subtle gradients, and dynamic background blurs.
- **Interactive Project Showcases**: Detailed case studies of my core projects (LifeFlow AI, StudyFlow AI, Naxora) complete with hover animations and immersive visuals.
- **Dynamic Bento Grid Layouts**: Academic and extracurricular achievements, including trophies and certificates, are displayed in a responsive bento-box grid structure.
- **Liquid Glass Tooltips & Modals**: Custom-built interactive bubbles with smooth pop-in animations (powered by GSAP) for viewing detailed award descriptions.
- **Fully Responsive**: Carefully optimized for seamless viewing across both desktop and mobile devices.

## 🛠️ Technology Stack

This project was built with modern web technologies focusing on performance and fluid animations:

- **Framework:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Vanilla CSS3 (Custom Properties, Flexbox, CSS Grid)
- **Animations:** GSAP (GreenSock Animation Platform)
- **Icons:** React Icons

## 🚀 Running the Project Locally

If you'd like to clone and run this portfolio on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SyedAbubakarSiddique/AbubakarSiddique_portfolio.git
   cd AbubakarSiddique_portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View in browser:**
   Open `http://localhost:5173` in your web browser.
